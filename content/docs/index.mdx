Selamat datang di dokumentasi **iPaymu API v2**. API ini memungkinkan Anda untuk mengintegrasikan layanan payment gateway iPaymu ke dalam aplikasi Anda.

## Base URLs

- **Production/Live**: `https://my.ipaymu.com`
- **Development/Sandbox**: `https://sandbox.ipaymu.com`

<Callout type="warn" title="Catatan Penting">
  * **VA (Virtual Account)** dan **API Key** dapat ditemukan di menu **Integrasi** pada dashboard
  iPaymu Anda. * VA dan API Key untuk mode **Sandbox/Development** **berbeda** dengan mode
  **Production/Live**. * Jika Anda mengalami kendala integrasi, silakan hubungi
  [support@ipaymu.com](mailto:support@ipaymu.com) atau via WhatsApp [di
  sini](https://wa.link/p8vnr7).
</Callout>

## Autentikasi

iPaymu API v2 menggunakan **autentikasi berbasis Signature**. Setiap request harus menyertakan header berikut:

| Header      | Deskripsi                                                                                                                                       |
| :---------- | :---------------------------------------------------------------------------------------------------------------------------------------------- |
| `va`        | Nomor Virtual Account Anda (dari Dashboard iPaymu).                                                                                             |
| `signature` | String signature yang dihasilkan untuk memverifikasi request.                                                                                   |
| `timestamp` | Timestamp saat ini (format: `YYYYMMDDHHmmss` atau ISO 8601 tergantung kebutuhan endpoint, biasanya `YYYYMMDDHHmmss` untuk pembuatan signature). |

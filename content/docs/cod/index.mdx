## Buat Transaksi COD

**URL**: `{{baseUrl}}/api/v2/payment/direct`
**Metode**: `POST`

### Request Body

| Parameter         | Tipe   | Wajib | Deskripsi                               |
| :---------------- | :----- | :---- | :-------------------------------------- |
| `name`            | string | Ya    | Nama pembeli.                           |
| `amount`          | number | Ya    | Total jumlah transaksi.                 |
| `paymentMethod`   | string | Ya    | Harus diisi `cod`.                      |
| `paymentChannel`  | string | Ya    | Kode channel pembayaran (misal: `rpx`). |
| `deliveryArea`    | string | Ya    | Kode pos pengiriman.                    |
| `deliveryAddress` | string | Ya    | Alamat lengkap pengiriman.              |
| `shipping`        | string | Ya    | Nama jasa pengiriman (misal: `RPX`).    |
| `shippingService` | string | Ya    | Tipe layanan pengiriman (misal: `REG`). |

Lihat [dokumentasi Direct Payment](/docs/payment/direct-payment) untuk daftar parameter lengkap.

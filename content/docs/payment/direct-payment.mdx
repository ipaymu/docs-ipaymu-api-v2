Endpoint ini digunakan untuk memproses pembayaran langsung (Direct Payment).

**Endpoint:** `POST https://my.ipaymu.com/api/v2/payment/direct`

## Headers

| Kunci          | Nilai              | Deskripsi                 |
| :------------- | :----------------- | :------------------------ |
| `Content-Type` | `application/json` |                           |
| `signature`    | `string`           | Signature yang dihasilkan |
| `va`           | `string`           | Nomor Virtual Account     |
| `timestamp`    | `string`           | Waktu request             |

## Parameter Body

| Parameter         | Tipe     | Wajib       | Deskripsi                         |
| :---------------- | :------- | :---------- | :-------------------------------- |
| `name`            | `string` | Ya          | Nama pembeli.                     |
| `phone`           | `string` | Ya          | No telepon pembeli.               |
| `email`           | `string` | Ya          | Email pembeli.                    |
| `amount`          | `number` | Ya          | Harga total.                      |
| `notifyUrl`       | `string` | Ya          | URL Notifikasi untuk menerima webhook dari iPaymu. (iPaymu akan mengirimkan parameter status dalam method POST ke URL ini ketika pengguna berhasil melakukan pembayaran). |
| `referenceId`     | `string` | Ya          | ID Referensi atau Nomor Transaksi merchant. |
| `paymentMethod`   | `string` | Ya          | Metode pembayaran. Nilai: `va`, `cstore`, `cod`, `qris`, `cc`, `paylater`, `ewallet`. |
| `paymentChannel`  | `string` | Ya          | Saluran berdasarkan metode pembayaran. Lihat rinciannya di bawah tabel. |
| `expired`         | `number` | Tidak       | Kustom masa berlaku dalam jam. **Catatan:** <br/>- BSI VA maks 3 jam <br/>- BRI VA maks 2 jam <br/>- BCA VA default 12 jam (tidak dikenakan custom) <br/>- Alfamart default 24 jam (tidak dikenakan custom) <br/>- QRIS default 5 menit (tidak dikenakan custom) |
| `comments`        | `string` | Tidak       | Catatan tambahan transaksi.       |
| `feeDirection`    | `string` | Tidak       | Kustom pengaturan pembebanan biaya: `MERCHANT` (admin fee ditanggung merchant) atau `BUYER` (ditanggung pembeli). |
| `escrow`          | `string` | Tidak       | Aktivasi layanan escrow dompet bersama (`true` / `false`). |
| `product[]`       | `array`  | Kondisional | Produk (wajib jika COD).          |
| `qty[]`           | `array`  | Kondisional | Kuantitas (wajib jika COD).       |
| `price[]`         | `array`  | Kondisional | Harga produk (wajib jika COD).    |
| `weight[]`        | `array`  | Kondisional | Berat barang dalam Kg (wajib jika COD). |
| `width[]`         | `array`  | Kondisional | Lebar dimensi barang dalam cm (wajib jika COD). |
| `height[]`        | `array`  | Kondisional | Tinggi dimensi barang dalam cm (opsional jika COD). |
| `length[]`        | `array`  | Kondisional | Panjang barang dalam cm (wajib jika COD). |
| `deliveryArea`    | `string` | Kondisional | Kode pos pengiriman (wajib jika COD). |
| `deliveryAddress` | `string` | Kondisional | Alamat pengiriman (wajib jika COD). |
| `shipping`        | `string` | Kondisional | Nama ekspedisi dari API Ongkir (wajib jika COD). |
| `shippingService` | `string` | Kondisional | Servis ekspedisi dari API Ongkir (wajib jika COD). |
| `pickupArea`      | `string` | Kondisional | Kode pos area penjemputan (wajib jika COD). |
| `account`         | `string` | Tidak       | ID Child account VA (opsional).   |
| `successUrl`      | `string` | Tidak       | URL rute sukes (URL *redirect* khusus Akulaku & Credit Card). |
| `cancelUrl`       | `string` | Tidak       | URL rute gagal (URL *redirect* khusus Akulaku & Credit Card). |

<details open>
<summary>Daftar <code>paymentChannel</code> yang tersedia berdasarkan <code>paymentMethod</code></summary>
<div className="pt-4">
| Metode Pembayaran (`paymentMethod`) | Daftar Saluran (`paymentChannel`) |
| :---------------------------------- | :-------------------------------- |
| **Virtual Account (`va`)**          | `bag`, `bca`, `bpd_bali`, `bni`, `cimb`, `mandiri`, `bmi`, `bri`, `bsi`, `permata`, `danamon`,`btn` |
| **Convenience Store (`cstore`)**    | `alfamart`, `indomaret`           |
| **Cash On Delivery (`cod`)**        | `rpx` (Kurir RPX)                 |
| **QRIS (`qris`)**                   | `mpm`                             |
| **Credit Card (`cc`)**              | `cc`                              |
| **Pay Later (`paylater`)**          | `akulaku`                         |
| **e-Wallet (`ewallet`)**            | `shopeepay`                       |
</div>
</details>

## Contoh Request

```json
  "name": "Buyer",
  "phone": "08123456789",
  "email": "buyer@mail.com",
  "amount": 10000,
  "notifyUrl": "https://mywebsite.com/callback",
  "expired": 24,
  "expiredType": "hours",
  "comments": "Payment for Order #123",
  "referenceId": "REF123456",
  "paymentMethod": "va",
  "paymentChannel": "bca",
  "product": ["Product A"],
  "qty": [1],
  "price": [10000],

  "width": [1],

  "length": [1],
  "deliveryArea": "76111",
  "deliveryAddress": "Jalan Raya No. 1"
}
```

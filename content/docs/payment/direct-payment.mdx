Endpoint ini digunakan untuk memproses pembayaran langsung (Direct Payment).

**Endpoint:** `POST https://my.ipaymu.com/api/v2/payment/direct`

## Headers

| Kunci          | Nilai              | Deskripsi                 |
| :------------- | :----------------- | :------------------------ |
| `Content-Type` | `application/json` |                           |
| `signature`    | `string`           | Signature yang dihasilkan |
| `va`           | `string`           | Nomor Virtual Account     |
| `timestamp`    | `string`           | Waktu request             |

## Parameter Body

| Parameter         | Tipe     | Deskripsi                         |
| :---------------- | :------- | :-------------------------------- |
| `name`            | `string` | Nama pelanggan                    |
| `phone`           | `string` | Nomor telepon pelanggan           |
| `email`           | `string` | Email pelanggan                   |
| `amount`          | `number` | Jumlah pembayaran                 |
| `notifyUrl`       | `string` | URL untuk notifikasi callback     |
| `expired`         | `number` | Waktu kedaluwarsa                 |
| `expiredType`     | `string` | Tipe kedaluwarsa (misal: "hours") |
| `comments`        | `string` | Komentar transaksi                |
| `referenceId`     | `string` | ID referensi unik                 |
| `paymentMethod`   | `string` | Metode pembayaran (misal: "va")   |
| `paymentChannel`  | `string` | Saluran pembayaran (misal: "bca") |
| `product`         | `array`  | Daftar produk                     |
| `qty`             | `array`  | Daftar kuantitas                  |
| `price`           | `array`  | Daftar harga                      |
| `weight`          | `array`  | Daftar berat                      |
| `width`           | `array`  | Daftar lebar                      |
| `height`          | `array`  | Daftar tinggi                     |
| `length`          | `array`  | Daftar panjang                    |
| `deliveryArea`    | `string` | Kode area pengiriman              |
| `deliveryAddress` | `string` | Alamat pengiriman                 |

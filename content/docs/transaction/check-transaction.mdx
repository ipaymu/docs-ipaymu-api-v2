Cek status dan detail dari transaksi tertentu.

## Endpoint

**URL**: `{{baseUrl}}/api/v2/transaction`
**Metode**: `POST`

## Request Headers

| Kunci          | Nilai              | Deskripsi                                                                |
| :------------- | :----------------- | :----------------------------------------------------------------------- |
| `Content-Type` | `application/json` |                                                                          |
| `signature`    | `{{signature}}`    | Signature yang dihasilkan. Lihat [Pembuatan Signature](/docs/signature). |
| `va`           | `{{va}}`           | Nomor Virtual Account Anda.                                              |
| `timestamp`    | `{{timestamp}}`    | Timestamp saat ini.                                                      |

## Request Body

| Parameter       | Tipe   | Wajib | Deskripsi                             |
| :-------------- | :----- | :---- | :------------------------------------ |
| `transactionId` | string | Ya    | ID Transaksi iPaymu yang ingin dicek. |
| `account`       | string | Tidak | VA iPaymu Anda (opsional).            |

## Contoh Request

```json
{
  "transactionId": "4719"
}
```

<Callout type="info" title="Kode Status Transaksi">
  * `0`: Pending (Menunggu Pembayaran) * `1`: Success (Berhasil) * `2`: Cancelled (Dibatalkan) *
  `3`: Refund (Dikembalikan) * `4`: Error * `5`: Failed (Gagal) * `6`: Success - Unsettled (Berhasil
  - Belum Settlement) * `7`: Escrow * `-2`: Expired (Kedaluwarsa)
</Callout>

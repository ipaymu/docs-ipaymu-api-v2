Check the status and details of a specific transaction.

## Endpoint

**URL**: `{{baseUrl}}/api/v2/transaction`
**Method**: `POST`

## Request Headers

| Key            | Value              | Description                                                       |
| :------------- | :----------------- | :---------------------------------------------------------------- |
| `Content-Type` | `application/json` |                                                                   |
| `signature`    | `{{signature}}`    | Generated signature. See [Signature Generation](/docs/signature). |
| `va`           | `{{va}}`           | Your Virtual Account Number.                                      |
| `timestamp`    | `{{timestamp}}`    | Current timestamp.                                                |

## Request Body

| Parameter       | Type   | Required | Description                         |
| :-------------- | :----- | :------- | :---------------------------------- |
| `transactionId` | string | Yes      | The iPaymu transaction ID to check. |
| `account`       | string | No       | Your iPaymu VA (optional).          |

## Example Request

```json
{
  "transactionId": "4719"
}
```

<Callout type="info" title="Transaction Status Codes">
  * `0`: Pending * `1`: Success * `2`: Cancelled * `3`: Refund * `4`: Error * `5`: Failed * `6`:
  Success - Unsettled * `7`: Escrow * `-2`: Expired
</Callout>

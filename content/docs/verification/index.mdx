Untuk mengaktifkan akun iPaymu Anda sepenuhnya dan mulai menerima pembayaran, Anda harus menyelesaikan proses verifikasi. Berikut adalah panduan langkah demi langkah untuk melakukan verifikasi di lingkungan Sandbox.

<Callout type="info" title="Prasyarat">
  Sebelum memulai, pastikan Anda memiliki data berikut: - **No. HP** yang aktif. - **Email** yang
  valid. - **KTP** (NIK wajib 16 angka). - **Foto Selfie** dengan KTP. - **Rekening Bank** (Nomor
  rekening dan bukti kepemilikan seperti buku tabungan atau screenshot m-banking). - **URL
  Website/Toko Online** yang valid dan dapat diakses publik.
</Callout>

<Steps>
<Step>

## Navigasi ke Halaman Verifikasi

Setelah berhasil [Register](https://sandbox.ipaymu.com/register) dan Login ke Dashboard:

1.  Cari tombol atau menu **Verifikasi Sekarang** di dashboard.
2.  Atau klik menu "Verifikasi" di sidebar dan klik tombol "Verifikasi Sekarang".

![Dashboard Verifikasi](/img/verifications/dashboard.png)

</Step>

<Step>

## Langkah 1: Informasi Pribadi

Isi data pribadi Anda dengan lengkap dan benar sesuai KTP.

![Langkah 1 - Informasi Pribadi](/img/verifications/step1-profile.png)

- **Nama Lengkap**: Sesuai KTP.
- **Alamat**: Alamat lengkap sesuai domisili.
- **NIK**: Wajib 16 digit angka.

Klik **Selanjutnya**.

</Step>

<Step>

## Langkah 2: Informasi Usaha

Isi detail mengenai bisnis atau toko online Anda.

![Langkah 2 - Informasi Usaha](/img/verifications/step2-business.png)

- **Nama Usaha/Toko**: Nama bisnis Anda.
- **URL Website**: Masukkan URL website atau toko online Anda yang valid (Harus benar).
  > <Callout type="warn">
  >   URL Website/Toko Online harus valid dan dapat diakses. Jika Anda belum memiliki domain
  >   sendiri, pastikan URL tersebut adalah URL yang sah.
  > </Callout>
- **Upload Dokumen**:
  - Untuk **Selfie**, Anda dapat menggunakan contoh gambar selfie.
  - Untuk file upload lainnya (KTP, NPWP jika ada), gunakan file gambar (misal: ktp-min.png).

Klik **Selanjutnya**.

</Step>

<Step>

## Langkah 3: Informasi Bank

Masukkan data rekening bank untuk penarikan dana.

![Langkah 3 - Informasi Bank](/img/verifications/step3-bank.png)

- **Nama Bank**: Pilih bank Anda.
- **No. Rekening**: Masukkan nomor rekening.
- **Nama Pemilik Rekening**: Harus sama dengan nama pendaftar.
- **Dokumen Pendukung**:
  - Jika tidak punya buku tabungan, Anda bisa mengunggah **screenshot mutasi rekening** atau halaman profil aplikasi **m-banking** yang menampilkan nomor rekening dan nama pemilik.

Klik **Selanjutnya**.

</Step>

<Step>

## Langkah 4: Dokumen Tambahan

![Langkah 4 - Dokumen Tambahan](/img/verifications/step4-document.png)

Unggah dokumen tambahan jika diminta, atau selesaikan proses jika sudah lengkap.

</Step>

</Steps>

## Catatan Penting

### Validasi Domain & IP

Pastikan website Anda sudah siap. Untuk pengaturan lebih lanjut mengenai whitelist IP dan domain, Anda bisa mengakses menu Integrasi:

- [Validasi Domain](https://sandbox.ipaymu.com/domain)
- [Validasi IP](https://sandbox.ipaymu.com/ip)

<Callout type="tip" title="IP Statis">
  Disarankan untuk menggunakan **IP Statis** untuk server Anda demi keamanan dan stabilitas
  transaksi. Jika Anda tidak memiliki IP Statis, silakan hubungi tim support kami di
  [support@ipaymu.com](mailto:support@ipaymu.com) untuk konsultasi solusi alternatif.
</Callout>

### Kesalahan Umum

- **NIK kurang dari 16 digit**: Pastikan NIK yang dimasukkan tepat 16 angka.
- **Foto buram**: Pastikan KTP dan Selfie dapat dibaca dengan jelas oleh sistem verifikasi.
- **Nama Rekening Berbeda**: Nama di rekening bank harus sesuai dengan nama verifikasi akun.

API ini berfungsi untuk mengambil detail profil sebuah akun iPaymu berdasarkan nomor Virtual Account
(VA).

## Endpoint

**URL**: `{{baseUrl}}/api/v2/profile`
**Method**: `POST`

| Lingkungan | URL                                         |
| :--------- | :------------------------------------------ |
| Sandbox    | `https://sandbox.ipaymu.com/api/v2/profile` |
| Production | `https://my.ipaymu.com/api/v2/profile`      |

## Request Header

Lihat [header & signature](/close-api) yang wajib disertakan pada setiap request Close API.

## Request Body

| Parameter | Tipe   | Wajib | Deskripsi          |
| :-------- | :----- | :---- | :----------------- |
| `account` | string | Ya    | Nomor VA iPaymu.   |

## Contoh Request

```json
{
  "account": "1234567890"
}
```

## Keterangan Parameter Response

| Field                    | Deskripsi                                            |
| :----------------------- | :-------------------------------------------------- |
| `Name`                   | Nama pemilik akun.                                   |
| `Email`                  | Email akun.                                          |
| `Phone`                  | No. HP akun.                                         |
| `ApiKey`                 | API Key akun.                                        |
| `Va`                     | Nomor Virtual Account.                              |
| `Gender`                 | Jenis kelamin (`L` = Laki-laki, `P` = Perempuan).   |
| `Nik`                    | Nomor Induk Kependudukan (KTP).                     |
| `IsMerchant`             | `true` jika akun adalah merchant.                   |
| `MerchantBalance`        | Saldo merchant.                                      |
| `MemberBalance`          | Saldo member.                                        |
| `VerificationStatus`     | Status verifikasi (lihat [Member Verification](/close-api/member-verification)). |
| `VerificationStatusDesc` | Deskripsi status verifikasi (mis. `VERIFIED`).      |
| `BankName` / `BankAccount` / `BankNumber` | Detail rekening bank akun.         |
| `RegisterDate`           | Tanggal pendaftaran akun.                           |

<Callout type="warn">
  Response dapat berisi data sensitif (mis. `ApiKey`, `Nik`). Pastikan data tidak dicatat di log dan
  hanya diproses pada sisi server yang aman.
</Callout>

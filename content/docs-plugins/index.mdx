## Download Plugin

Kunjungi website resmi iPaymu di [https://ipaymu.com/](https://ipaymu.com/), lalu pilih **Doc API -> Download Instant Plugin**.

![Download Plugin](/img/plugins/image1.jpeg)

Pilih plugin iPaymu sesuai yang anda butuhkan. Di iPaymu, kami menyediakan plugin untuk **WP WooCommerce**, **OpenCart 3.x**, **Prestashop 1.x**, **WHMCS**, dan **Moodle**.

![Pilih Plugin](/img/plugins/image2.jpeg)

## WP WooCommerce

Ikuti cara dibawah untuk install plugin WP WooCommerce:

1. **Login** ke dashboard Wordpress anda, kemudian pilih menu **Plugins -> Tambah Baru (Add New)**.

![Add New Plugin](/img/plugins/image3.jpeg)

2. Klik **Unggah Plugin (Upload Plugin)**, dan unggah plugin yang anda download dalam bentuk `.zip`.

![Upload Plugin](/img/plugins/image4.jpeg)
![Upload Zip](/img/plugins/image5.jpeg)

3. Jika plugin berhasil terpasang, plugin akan terlihat di menu **Plugin Terpasang**, lalu aktifkan plugin.

![Activate Plugin](/img/plugins/image6.jpeg)

4. Setelah itu, pilih menu **WooCommerce -> Pengaturan (Settings)**.

![WooCommerce Settings](/img/plugins/image7.jpeg)

5. Pilih tab menu **Pembayaran (Payments)**, aktifkan metode pembayaran **iPaymu Payment Gateway** dan klik tombol **kelola (manage)** untuk konfigurasikan plugin.

![Manage Payment](/img/plugins/image8.jpeg)

6. Isi form yang muncul sesuai data toko anda. Untuk **API Key** anda, dapat dilihat di dashboard iPaymu.

![Configure Plugin](/img/plugins/image9.jpeg)

7. Jika toko anda dalam mode sandbox, centang checkbox **Mode Sandbox/Development**.

![Sandbox Mode](/img/plugins/image10.jpeg)

8. Jika anda ingin melakukan custom untuk halaman **“Terimakasih”** (halaman yang muncul setelah checkout), anda tinggal memasukan URL link halaman tersebut disini.

![Thank You Page](/img/plugins/image11.jpeg)

9. Setelah selesai, klik **Simpan Perubahan**.

## Opencart

### Cara 1 (First Way)

1. **Login** dahulu ke halaman dashboard Opencart anda.

![Login Opencart](/img/plugins/image15.jpeg)

2. Setelah login, klik menu **Extensions -> Installer** lalu klik tombol **Upload**.

![Opencart Installer](/img/plugins/image16.jpeg)

3. Unggah plugin yang anda download dalam bentuk `.zip`.

![Upload Opencart](/img/plugins/image17.jpeg)

4. Jika plugin berhasil terpasang, akan muncul pesan sukses.

![Success Opencart](/img/plugins/image18.png)

5. Kemudian klik **Extensions -> Extensions**, pilih filter **Payments**. Cari **iPaymu**, lalu klik tombol **Install** (icon hijau).

![Install iPaymu](/img/plugins/image19.jpeg)

6. Setelah install, klik tombol **Edit** (icon biru).

![Edit iPaymu](/img/plugins/image20.jpeg)

7. Isi form dengan username dan Api Key yang ada di dashboard akun iPaymu anda. Lalu klik **Save**.

![Configure Opencart](/img/plugins/image21.jpeg)

## Prestashop

1. **Login** dahulu ke halaman dashboard Prestashop anda.

![Login Prestashop](/img/plugins/image22.jpeg)

2. Klik menu **Modules -> Module Manager** lalu klik tombol **Upload a Module**.

![Module Manager](/img/plugins/image23.jpeg)

3. Unggah plugin yang anda download dalam bentuk `.zip`.

![Upload Prestashop](/img/plugins/image24.jpeg)

4. Jika plugin berhasil terpasang, akan muncul pesan sukses. Klik **Configure**.

5. Isi form dengan username dan Api Key yang ada di dashboard akun iPaymu anda. Lalu klik **Save**.

## WHMCS

**WHMCS** (Web Host Manager Complete Solution) adalah platform otomatisasi bisnis hosting yang banyak digunakan oleh Hosting Provider, Domain Registrar, Web Agency, dan Developer.

1. Download plugin WHMCS dari halaman [Plugin iPaymu](https://ipaymu.com/id/plugin-download/).

![Download Plugin WHMCS](/img/plugins/whmcs/image1.webp)

2. Setelah diekstrak, copy folder `callback/` ke `/modules/gateways/callback/` dan file `ipaymu/` serta `ipaymu.php` ke `/modules/gateways/`.

3. Masuk ke **Settings** → **Apps & Integrations**, pilih **Browse** → **Payments**.

![Settings Apps & Integrations](/img/plugins/whmcs/image2.webp)

![Browse Payments](/img/plugins/whmcs/image3.webp)

4. Cari **iPaymu Direct Payment**, klik **Activate**. Setelah aktif, klik **Manage**.

![Aktivasi iPaymu Direct Payment](/img/plugins/whmcs/image4.webp)

![Status Module Aktif](/img/plugins/whmcs/image5.webp)

5. Masukkan **Environment** (`Sandbox` atau `Production`), **VA Number**, dan **API Key**. Klik **Save Changes**.

![Pengaturan Environment iPaymu](/img/plugins/whmcs/image6.webp)

6. Atur **Product Group**, **Product**, dan **Pricing** sesuai kebutuhan.

![Product Group dan Product](/img/plugins/whmcs/image8.webp)

7. Customer memilih produk → **Order Now** → **Checkout** → pilih **iPaymu Direct Payment** → **Complete Order**.

![Pemilihan Metode Pembayaran](/img/plugins/whmcs/image12.webp)

8. Customer diarahkan ke halaman pembayaran iPaymu. Setelah sukses, status invoice menjadi **PAID**.

![Status Invoice PAID](/img/plugins/whmcs/image16.webp)

Untuk panduan lengkap, kunjungi halaman [Plugin WHMCS](/docs-plugins/whmcs).

## Moodle

**Plugin Moodle** adalah komponen tambahan untuk platform e-learning Moodle yang memungkinkan integrasi pembayaran kursus melalui iPaymu.

1. Download plugin Moodle dari halaman [Plugin iPaymu](https://ipaymu.com/id/plugin-download/).

2. Login sebagai **Admin** ke situs Moodle, buka **Site administration** → **Plugins** → **Install plugins**.

3. Upload file `.zip` plugin, klik **Install plugin from the ZIP file**, lalu klik **Continue**.

4. Klik **Upgrade Moodle database now**. Setelah selesai, atur **Environment** (`Sandbox` atau `Production`), masukkan **API Key** dan **VA Number**.

5. Aktifkan **Plugin Enrolment** iPaymu Payment. Pastikan plugin aktif dengan ikon **mata** yang menyala biru.

6. Buat course baru melalui **New Course**, isi keterangan, lalu klik **Save and display**.

7. Klik tab **Participants** → **Enrolment methods** → **Add method** → pilih **iPaymu Payment**.

8. Isi deskripsi dan harga course, lalu klik **Add method**.

9. Ketika user mengakses course, akan muncul halaman pembayaran. Klik **Pay via iPaymu**.

10. Setelah pembayaran berhasil, user dapat langsung mengakses isi materi course.

Untuk panduan lengkap, kunjungi halaman [Plugin Moodle](/docs-plugins/moodle).
